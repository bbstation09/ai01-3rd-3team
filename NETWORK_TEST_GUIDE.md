# 🎫 티켓 예매 시스템 테스트 가이드

## 접속 방법

### 서버 주소
```
http://192.168.14.199:8000
```

### 접속 조건
- ✅ **같은 네트워크(WiFi)에 연결**되어 있어야 합니다
- ✅ 브라우저에서 위 주소 입력

---

## 테스트 순서

| 단계 | 설명 |
|------|------|
| 1️⃣ | 위 주소로 접속 |
| 2️⃣ | **회원가입** 또는 **로그인** |
| 3️⃣ | 공연 선택 |
| 4️⃣ | 대기열 통과 |
| 5️⃣ | 캡챠 입력 |
| 6️⃣ | 구역 선택 → 좌석 선택 |
| 7️⃣ | 할인권종 → 배송방법 → 결제수단 선택 |
| 8️⃣ | 예매 완료 |

---

## 주의사항

1. **자연스럽게 사용하세요** - 마우스 움직임, 클릭 등이 기록됩니다
2. **여러 좌석을 살펴보세요** - 탐색 행동도 중요한 데이터입니다
3. **동시 접속 가능** - 10명 이상 동시 테스트 OK

---

## 접속 안 될 때

1. WiFi 연결 확인
2. 주소 정확히 입력: `http://192.168.14.199:8000`
3. 서버 관리자에게 문의

---
